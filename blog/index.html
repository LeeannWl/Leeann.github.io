CSS选择器

作用：用来指定网页上我们想要样式化的HTML元素

选择器的种类

类型选择器

h1 { }

ID选择器（#）和CLASS选择器（.）

#unique { }
.box { }

通配选择器

* {  }

标签属性选择器

a[title] { }

伪类与伪元素

p:first-child { }
p::first-line { }

后代选择器和子代选择器

article p
article > p

相邻兄弟选择器和通用兄弟选择器

h1 + p
h1 ~ p

选择器的优先级

一般而言，选择器越特殊，它的优先级越高。也就是选择器指向的越准确，它的优先级就越高。

!important > [id > class > tag]
